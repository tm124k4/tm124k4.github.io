bcolor 

bcolor(colorB,easetype,duration)

概要

E().bcolor()は、E()で取得したエレメントの背景色を任意のものに変更するメソッドです。

引数 内容 型 必須
colorB 変更先の色({r,g,b})
変更先の色コード		 Object
String		 yes
easetype イージングタイプ String no
duration 速度 Number no

解説

標準のjavascriptである要素の背景色や文字色を変更したい場合、プロパティに直接入力するのが一般的です。

動的に色を変更、例えばフェードしながら赤色から青色に変更させるような場合については、ライブラリなしで行う場合、通常はrequestAnimationFrameを使うことが考えられますが、１から実装する場合は複雑かつコード量も多くなります。

kentaur.jsでは、以下の通りそれぞれ１行にまとめて書くことができます。


E(".target").bcolor({r:0,g:255,b:0,a:1},"inoutquad",3000)

上記コードの場合、.target要素を3000ms秒かけてinoutquadのイージングタイプで背景色を緑色にします。そのほか、以下のような記法も可能です。


E(".target").bcolor("#00ff00","inoutquad",3000)         //RGB 16進数表記(RRGGBB)
E(".target").bcolor("#00ff00ff","inoutquad",3000)       //RGBA 16進数表記(RRGGBBAA)
E(".target").bcolor("#0f0","inoutquad",3000)            //RGB 16進数表記 16段階(RGB)
E(".target").bcolor("#ff","inoutquad",3000)             //16進数表記(グレースケール)
E(".target").bcolor("lime","inoutquad",3000)            //CSS3基本色キーワード表記
E(".target").bcolor("lawngreen","inoutquad",3000)       //CSS3拡張色キーワード表記
E(".target").bcolor("rebbeccapurple","inoutquad",3000)  //CSS4のキーワード表記
E(".target").bcolor("rgb(0,255,0)","inoutquad",3000)    //CSS rgb
E(".target").bcolor("rgba(0,255,0,1)","inoutquad",3000) //CSS rgba

rgbは0から255までの整数、a（透明度）は1（透明度0%）から0（透明度100%）までの少数で表されます。
