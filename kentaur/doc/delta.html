delta 

delta(x,y,w,h,d)

概要

E().delta()は、spanによる三角形を描くメソッドです。描画されたspanはE()で取得したエレメントの下に追加されます。描かれる三角形は正三角形または二等辺三角形であり、頂点は画面上に対して通常真上（0度）から始まります。

引数 内容 型 必須
x 左上x座標 Number
String		 yes
y 左上y座標 Number
String		 yes
w 横幅 Number
String		 yes
h 縦幅 Number
String		 no
d 角度 Number no

解説

通常、Javascriptを用いてブラウザ上で三角形を描くにはcanvasやSVGがよく使われます。しかし、どちらもデフォルトでは描画する関数は用意されていません。

//canvasの場合
var cv=document.getElementById("canvas1"),
ctx=cv.getContext('2d');
ctx.fillStyle='red';
ctx.moveTo(75,50);
ctx.lineTo(100,75);
ctx.lineTo(100,25);
ctx.fill();

//SVGの場合
var sv=document.getElementById("svg1"),
path=document.createElementNS("http://www.w3.org/2000/svg","path");
path.setAttribute('d','M50 0 L0 100 L100 100 Z');
path.setAttribute('style','fill:red');
sv.appendChild(path);

canvasで三角形を描く場合、ライブラリを使わない場合は描画に必要となる座標の計算を行う、あるいは、3点の座標を取得して、moveToとlineToを合計3回実行して一つなぎの三角形パスを作成する必要があります。

角度を指定する場合、ライブラリなしで実行しようとするならばもっと複雑なコードになります。また、canvasの場合は一度描画した後に変形などを行う事は簡単ではありません。

SVGで三角形を描く場合も各地点の計算は必須です。また、親要素がsvgであることが前提で、少なくとも２つ以上のタグを必要とします。

kentaur.jsでは生成される新たなspan要素に対して自動的に必要なCSSスタイルを適用し、以下の通り１行にまとめて書くことができます。

E("div#kentaur1").delta(50,50,50,50,45)

上記の場合、横50px,縦50pxの座標から横50px,縦50pxの四角形の範囲内で45度の方向を向いた正三角形が描かれます。縦幅と横幅が異なる場合は二等辺三角形が描かれます。また、縦幅を省略した場合は自動的に横幅のサイズで描画されます。また、各引数は文字列型でも入力可能で、%のようにCSSで有効な記法を用いて表現することも可能です。

現状、１色塗りつぶしのみ対応しています。
