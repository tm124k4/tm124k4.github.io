hex 

hex(x,y,w,h)

概要

E().hex()は、spanによる六角形（ヘックス）を描くメソッドです。描画されたspanはE()で取得したエレメントの下に追加されます。

引数 内容 型 必須
x 左上x座標 Number
String		 yes
y 左上y座標 Number
String		 yes
w 横幅 Number
String		 yes
h 縦幅 Number
String		 no

解説

通常、Javascriptを用いてブラウザ上で六角形を描くにはcanvasやSVGがよく使われます。しかし、どちらもデフォルトで描画する関数は用意されていません。

canvasで六角形を描く場合、ライブラリを使わない場合は描画に必要となる座標の計算を行い、lineToを6回実行して一つなぎの六角形パスを作成する必要があります。また、canvasの場合は一度描画した後に変形などを行う事は簡単ではありません。

SVGで六角形を描く場合も各地点の計算は必須です。また、親要素がsvgであることが前提で、少なくとも２つ以上のタグを必要とします。

kentaur.jsでは生成される新たなspan要素に対して自動的に必要なCSSスタイルを適用し、以下の通り１行にまとめて書くことができます。

E("div#kentaur1").box(50,50,50,50)

上記の場合、横50px,縦50pxの座標から横50px,縦50pxの範囲内で六角形を描きます。また、縦幅を省略した場合は自動的に横幅のサイズで描画されます。また、各引数は文字列型でも入力可能で、%のようにCSSで有効な記法を用いて表現することも可能です。
