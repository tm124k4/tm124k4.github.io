arc 

arc(x,y,w,h)

概要

E().arc()は、spanによる円を描くメソッドです。描画されたspanはE()で取得したエレメントの下に追加されます。

引数 内容 型 必須
x 左上x座標 Number
String		 yes
y 左上y座標 Number
String		 yes
w 横幅 Number
String		 yes
h 縦幅 Number
String		 yes 
E("body");
E(document.body);
E("div#target");

解説

通常、Javascriptを用いてブラウザ上で円を描くにはcanvasやSVGがよく使われます。

//canvasの場合
var cv=document.getElementById("canvas1"),
ctx=cv.getContext('2d');
ctx.beginPath();
ctx.arc(100,100,50,0,2*Math.PI)
ctx.closePath();
ctx.fill();

//SVGの場合
var sv=document.getElementById("svg1"),
circle=document.createElementNS("http://www.w3.org/2000/svg","circle");
circle.setAttribute('cx',50);
circle.setAttribute('cy',50);
circle.setAttribute('r', 50);
circle.setAttribute('fill','red');
circle.setAttribute('stroke','black');
sv.appendChild(circle);

canvasでは半円やもっと複雑な円の形状も表現可能ですが、一度描画した図形を後から変形・移動させることが難しいほか、ライブラリなしではシンプルな円を表示したい時に数行のコードを必要とします。

SVGで円を描く場合は、canvasよりももう少し簡単にはなりますが、親要素がsvgであることが前提です。また、少なくとも２つ以上のタグを必要とします。

kentaur.jsでは生成される新たなspan要素に対して自動的に必要なCSSスタイルを適用し、以下の通り１行にまとめて書くことができます。

E("div#kentaur1").arc(50,50,50,50)

上記の場合、横50px,縦50pxの座標から横50px,縦50pxの四角形の範囲内で円を描きます。縦横の幅が長方形になる場合は楕円になります。また、各引数は文字列型でも入力可能で、%のようにCSSで有効な記法を用いて表現することも可能です。
