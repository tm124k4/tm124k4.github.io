E()

E(element)

引数 内容 型 必須
element 取得したい要素のCSSセレクタ
またはElementそのもの		 String
Object		 yes
nth 取得したい要素の番号 Number no

E()は、Kentaur.jsにおけるHTMLから任意の要素を取得するためのセレクタです。CSSセレクタで取得したい要素を選択します。

E("body");
E(document.body);
E("div#target");
E(".target",2)

javascript標準の関数であるdocument.querySelectorAllと使い方はほとんど同じですが、返り値にはエレメントそのものではなく、kentaurの様々なメソッドを内包したオブジェクト、すなわち取得した要素の子要素として線を描画したり、取得した要素自体をアニメーション移動させるための様々な関数が渡されます。
