fadeout 

fadeout(easetype,duration,reverse)

概要

E().fadeout()は、取得した要素をフェードアウトさせるメソッドです。

引数 内容 型 必須
easetype イージングタイプ
または時間(ms)		 String
Number		 yes
duration 時間(ms) Number no
reverse 逆転 Boolean no

解説

通常、Javascriptを用いて要素をフェードインもしくはフェードアウトさせる場合、CSSで予め対象の要素のopacityに対してtransitionを指定しておく必要があります。動的に生成された要素の場合、その要素のstyleにtransitionを追加しておく必要があります。

/* CSS */
.target {
    transition: opacity 3s;
}

/* Javascript */
document.querySelector(".target").style.opacity = 0;

kentaur.jsでは以下のような書き方でまとめることができます。

E("div#kentaur1").fadeout(1000)

また、イージングにも対応しており、第1引数に希望のイージングタイプ(String)を指定しておくとイージングに沿ってフェードアウトさせることができます。

サンプルを見る


    function main(){
    
        E("body").box(200,50,100,100);
        E(".box").fadeout("incirc",10000);

        E("body").hex(350,50,100,100);
        E(".hex").fadein("incirc",10000)

    }       
    window.onload=main;

第3引数はライブラリ内での処理の共通化のために用意されているもので、trueにした場合は、フェードインを行います。
